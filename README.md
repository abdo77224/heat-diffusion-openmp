# Heat Diffusion Simulation (Serial vs OpenMP)

## Description
This project simulates heat diffusion on a 2D grid using the finite difference method.

## Versions
- Serial implementation
- OpenMP parallel implementation

## Results
- Serial: ~11.77s
- OpenMP: ~3.80s
- Speedup: ~3x

## Technologies
- C++
- OpenMP
- Python (for visualization)
